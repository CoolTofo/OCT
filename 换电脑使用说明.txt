十月文化 / 千帧万象 - 纯净便携版

Windows 换电脑使用：

1. 解压整个文件夹。
2. 双击“启动服务.bat”。
3. 浏览器打开后，在左下角“设置 API”里填写自己的 API Key。

通常不需要重新配置 Python 环境：

- 已包含 python 文件夹，可直接运行。
- 已包含 packages 文件夹和 requirements.txt，缺依赖时可双击“安装依赖.bat”。
- 已保留 python.zip，遇到网盘或拷贝丢失 python 文件夹时，可以解压它恢复内置 Python。

本地 ComfyUI：

- 默认后端地址是 127.0.0.1:8188。
- 如果新电脑上的 ComfyUI 不是这个端口，请在软件左下角“ComfyUI 设置”里修改。

纯净版应清理：

- API 密钥
- 生成历史
- 输入参考图
- 输出图片/视频
- 画布草稿
- 对话记录
- Python 缓存
- 工作辅助记录
